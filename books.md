# Books

Technical books and references I use for AI systems, production engineering, and interview preparation.

<section class="content-grid">
  <article class="page-card">
    <h3><a href="https://amzn.to/40orUvs">AI Engineering</a></h3>
    <p><strong>O’Reilly, 2025</strong></p>
    <p>Practical reference for building applications on top of foundation models, including model selection, evaluation, deployment, and product patterns.</p>
  </article>
  <article class="page-card">
    <h3><a href="https://www.amazon.com/Designing-Machine-Learning-Systems-Production-Ready/dp/1098107969">Designing Machine Learning Systems</a></h3>
    <p><strong>O’Reilly, 2022</strong></p>
    <p>System-design view of ML in production: data pipelines, features, training loops, model monitoring, reliability, and iteration under real-world constraints.</p>
  </article>
  <article class="page-card">
    <h3><a href="https://huyenchip.com/ml-interviews-book/">Machine Learning Interviews</a></h3>
    <p><strong>2021</strong></p>
    <p>Interview preparation resource covering machine learning fundamentals, system design, practical judgment, and hiring-process expectations.</p>
  </article>
</section>
